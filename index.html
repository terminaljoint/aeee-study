Calculus — Limits, Differentiation, Integration
9–10 Qs🔥 Top▼
Priority
5/5 — Highest mark topic in all of Maths. Do this first.
Limits
lim(x→0) sinx/x = 1Also: lim tanx/x=1 and lim(1−cosx)/x²=½. These three appear every paper.
lim(x→0)(1+x)^(1/x) = eDefinition of e. Variant: lim(1+1/n)ⁿ=e as n→∞
lim(x→a)(xⁿ−aⁿ)/(x−a) = naⁿ⁻¹Use when you see (xⁿ−aⁿ) in numerator directly
Differentiation
d/dx(xⁿ) = nxⁿ⁻¹Power rule — used in almost every differentiation question
d/dx(uv) = u'v+uv' · d/dx(u/v) = (u'v−uv')/v²Product rule · Quotient rule
Chain: dy/dx = (dy/du)·(du/dx)For composite functions — most common type in AEEE
d/dx(sinx)=cosx · d/dx(cosx)=−sinxAlso: d/dx(tanx)=sec²x · d/dx(eˣ)=eˣ · d/dx(lnx)=1/x
Integration
∫xⁿdx = xⁿ⁺¹/(n+1)+CPower rule for integration (n≠−1)
∫sinx dx=−cosx+C · ∫cosx dx=sinx+CAlso: ∫eˣdx=eˣ+C · ∫(1/x)dx=ln|x|+C
∫1/(x²+a²)dx = (1/a)tan⁻¹(x/a)+CRecognize this form — appears very often
Area = ∫ₐᵇ |f(x)| dxUse |f(x)| if curve dips below x-axis in [a,b]
⚡ Practice MCQ
d/dx of x³·eˣ is:
A) 3x²eˣ
B) eˣ(x³+3x²)
C) x³eˣ
D) 3x²
Product rule: u=x³, v=eˣ. u'=3x², v'=eˣ. Answer = 3x²·eˣ + x³·eˣ = eˣ(x³+3x²).
⚠ Common Traps
  • d/dx(cosx) = −sinx (negative!). Forgetting this costs marks every year.
  • Forgetting +C in indefinite integrals — AEEE includes "which is correct" options that differ only by +C.
  • Area below x-axis: split the integral and take |value|. Raw integral gives wrong sign.
AEEE shortcut: if numerator = derivative of denominator → integral = ln|denominator|+C. Spot in 2 seconds.

Algebra — Matrices, Quadratics, AP/GP, Binomial
8–10 Qs🔥 Top▼
Priority
5/5 — Equal to Calculus in marks. Second topic to master.
Matrices & Determinants
|A| 2×2 = ad−bcMatrix [[a,b],[c,d]]. For 3×3: expand along first row with alternating signs.
A⁻¹ = adj(A)/|A| (only if |A|≠0)(AB)⁻¹=B⁻¹A⁻¹. Order reverses for inverse of product.
Quadratic equations
x = (−b ± √(b²−4ac)) / 2aQuadratic formula. D=b²−4ac: D>0 real distinct, D=0 equal, D<0 complex.
Sum of roots = −b/a · Product = c/aFor ax²+bx+c=0. Sum is −b/a NOT +b/a (negative sign!)
AP · GP · Sequences
AP: Tₙ=a+(n−1)d · S=n/2[2a+(n−1)d]Σn=n(n+1)/2 · Σn²=n(n+1)(2n+1)/6 · Σn³=[n(n+1)/2]²
GP: Tₙ=arⁿ⁻¹ · S=a(rⁿ−1)/(r−1)Infinite GP (|r|<1): S∞=a/(1−r) — very commonly tested
Binomial T(r+1) = C(n,r)·aⁿ⁻ʳ·bʳMiddle term: if n even → T(n/2+1). Find r from power of b in the term.
⚡ Practice MCQ
Sum of roots of 2x²−5x+3=0:
A) −5/2
B) 5/2
C) 3/2
D) 5
Sum = −b/a = −(−5)/2 = 5/2. Product = c/a = 3/2. The trap: −(−5) = +5, not −5.
⚠ Common Traps
  • Sum of roots = −b/a NOT +b/a. Negative sign is the most missed point.
  • Infinite GP: valid ONLY when |r|<1. Always check this condition first.
🧠 Memory Anchor

Quadratic roots: Sum=−b/a, Product=c/a. Say "Soup = −b/a, Cup = c/a" to remember.

Coordinate Geometry — Lines, Circles, Conics
5–6 Qs▼
Priority
4/5 — Full marks if you know standard forms cold.
Lines
m = (y₂−y₁)/(x₂−x₁)Forms: y−y₁=m(x−x₁) · y=mx+c · x/a+y/b=1
Dist(point to line) = |ax₁+by₁+c|/√(a²+b²)For line ax+by+c=0. Parallel: m₁=m₂. Perp: m₁m₂=−1.
tanθ = |m₁−m₂|/|1+m₁m₂|Acute angle between two lines — AEEE asks this directly
Circles
(x−h)²+(y−k)²=r² · Centre(h,k), radius rGeneral: x²+y²+2gx+2fy+c=0 → centre(−g,−f), r=√(g²+f²−c)
Conics
Parabola y²=4ax: focus(a,0), directrix x=−aLatus rectum=4a. Vertex at origin.
Ellipse x²/a²+y²/b²=1: foci(±c,0), c²=a²−b²e=c/a<1. If b>a → major axis on y-axis.
Hyperbola x²/a²−y²/b²=1: asymptotes y=±(b/a)xc²=a²+b². e=c/a>1.
⚡ Practice MCQ
Centre of circle x²+y²−6x+4y−12=0:
A) (−3,2)
B) (3,−2)
C) (6,−4)
D) (3,2)
2g=−6→g=−3; 2f=4→f=2. Centre=(−g,−f)=(3,−2). Signs flip!
⚠ Common Traps
  • General circle centre = (−g,−f) NOT (+g,+f). Signs flip — most common wrong answer option.
  • Ellipse: if b>a, vertices are on y-axis, NOT x-axis. Read the equation carefully.

Vectors, Probability & Binomial Distribution
5–7 Qs▼
Priority
4/5 — Probability appears every paper. Vectors are direct recall.
Vectors
|a|=√(x²+y²+z²) · â=a/|a|Magnitude · Unit vector. Always divide by magnitude to unitize.
a·b=|a||b|cosθ=x₁x₂+y₁y₂+z₁z₂Dot product. a⊥b ↔ a·b=0. a∥b ↔ a×b=0.
|a×b|=|a||b|sinθCross product magnitude = area of parallelogram formed by a and b
Probability
P(A)=n(A)/n(S) · P(A')=1−P(A)P(A∪B)=P(A)+P(B)−P(A∩B). Mutually exclusive: P(A∩B)=0.
P(A|B)=P(A∩B)/P(B)Bayes: P(A|B)=P(B|A)·P(A)/P(B). Independent: P(A∩B)=P(A)·P(B).
Binomial P(X=r)=C(n,r)pʳqⁿ⁻ʳMean=np · Variance=npq · SD=√(npq)
⚠ Common Traps
  • "At least 1" = 1−P(none). Never add individual cases — that's only for mutually exclusive events.
  • Independent ≠ Mutually exclusive. Independent: P(A∩B)=P(A)P(B). Mutually exclusive: P(A∩B)=0.

sin75° = ?

🧠 Memory Anchor — sin values 0°→90°

Permutations & Combinations
3–4 QsNEW▼
Priority
4/5 — Direct substitution. Quick 3–4 marks.
nPr=n!/(n−r)! · nCr=n!/[r!(n−r)!]P = arrangement (order matters). C = selection (order doesn't matter).
nCr=nC(n−r) · nC0=nCn=1Symmetry: C(10,7)=C(10,3). Use to simplify large numbers.
Repeats: n!/p!q!r! · Circular: (n−1)!Necklace (can flip): (n−1)!/2. "At least 1" = Total−(none).
⚡ Practice MCQ
5 people seated around a circular table. Ways = ?
A) 120
B) 24
C) 60
D) 5
Circular = (n−1)! = 4! = 24. One person is fixed as reference point — reduces n! to (n−1)!.
⚠ Common Traps
  • Circular = (n−1)!, NOT n!. This wrong option appears in every paper.
  • "At least 2" = Total − (none) − (exactly 1). Don't stop at subtracting just "none".

Statistics & Mathematical Reasoning
2–3 QsNEW▼
Priority
3/5 — Reasoning = zero calculation. Pure logic = free marks.
Mean=Σfx/Σf · Var σ²=Σfx²/Σf−(x̄)²σ=√Variance. CV=(σ/x̄)×100 for comparison.

Contrapositive of "P→Q" = "¬Q→¬P" — same truth value as original.

Converse "Q→P" and Inverse "¬P→¬Q" are NOT equivalent to original.

p∧q: true only when BOTH true. p∨q: false only when BOTH false.

¬(all X are Y) = "some X are not Y". ¬(some X are Y) = "no X are Y".

⚡ Practice MCQ

A) If no rain, match not cancelled
B) If match not cancelled, it didn't rain
C) If match cancelled, it rained
D) If it rains, match not cancelled
Contrapositive of P→Q is ¬Q→¬P. ¬Q="match not cancelled", ¬P="it didn't rain" → "If match not cancelled, it didn't rain."

Modern Physics — Photoelectric, Bohr, Nuclear
4–5 Qs🔥 Top▼
Priority
5/5 — Half-life + Bohr appear in EVERY AEEE paper. Master first.
E=hf=hc/λ · KE_max=hf−φ · eV₀=KE_maxPhoton energy. Einstein's photoelectric equation. Stopping potential.
λ_dB=h/mv · rₙ=0.529n²/Z Å · Eₙ=−13.6Z²/n² eVde Broglie wavelength. Bohr orbit radius. Bohr energy level.
N=N₀(1/2)^(t/T½) · λ_d=0.693/T½Radioactive decay. After n half-lives: N=N₀/2ⁿ. Activity A=λN.
BE = Δm × 931.5 MeV/amuBinding energy. Higher BE/nucleon = more stable nucleus.
⚡ Practice MCQ
Radioactive substance, half-life 10 days. Fraction remaining after 30 days:
A) 1/4
B) 1/8
C) 1/2
D) 1/16
30/10=3 half-lives. (1/2)³=1/8. Fraction REMAINING = 1/8. Fraction DECAYED = 7/8. Don't confuse the two.
⚠ Common Traps
  • Increasing intensity in photoelectric effect doesn't change KE_max — only frequency does. AEEE loves this conceptual trap.
  • Higher orbit = higher (less negative) energy. E₁
  • Half-life question: is it asking fraction remaining or fraction decayed? Always check.

Electrostatics & Current Electricity
3–4 Qs🔥 Top▼
Priority
5/5 — Circuits appear every single paper.
F=kq₁q₂/r² · E=kq/r² · V=kq/rk=9×10⁹. F∝1/r² (double distance = 1/4 force).
C=Q/V · Series: 1/C=Σ1/Cᵢ · Parallel: C=ΣCᵢEnergy=½CV²=Q²/2C. Capacitors in SERIES: smaller. OPPOSITE of resistors!
V=IR · P=VI=I²R=V²/RR_series=ΣRᵢ. 1/R_parallel=Σ(1/Rᵢ). Resistors in series: larger.
KCL: ΣI_in=ΣI_out · KVL: ΣV=0Wheatstone bridge: P/Q=R/S when balanced (no current through galvanometer).
⚡ Practice MCQ
4μF and 6μF in series. Equivalent capacitance:
A) 10μF
B) 2.4μF
C) 5μF
D) 24μF
1/C=1/4+1/6=5/12 → C=12/5=2.4μF. Capacitors in series combine like resistors in parallel.
⚠ Common Traps
  • Capacitors in SERIES → smaller (like resistors in parallel). This reversal catches everyone.
  • KVL sign: traversing with current through resistor = voltage DROP (−IR). Against = +IR.
🧠 Memory Anchor

Capacitors opposite to Resistors: C in Series = smaller (like R in parallel). C in Parallel = larger (like R in series).

Electromagnetic Induction & AC Circuits
3–4 Qs▼
Priority
4/5 — LCR resonance + RMS tested every paper.
EMF=−dΦ/dt · Φ=BAcosθ · EMF_motional=BlvFaraday's law. Lenz's law: induced EMF opposes the change causing it.
I_rms=I₀/√2 · V_rms=V₀/√2RMS ≠ Average. Average = 2V₀/π ≈ 0.637V₀. RMS ≈ 0.707V₀.
X_L=2πfL · X_C=1/2πfC · Z=√(R²+(X_L−X_C)²)Resonance: X_L=X_C, f₀=1/2π√(LC), Z=R (minimum), I=maximum.
cosφ=R/Z · P=V_rms·I_rms·cosφPower factor. At resonance: cosφ=1, maximum power.
⚠ Common Traps
  • At resonance: Z is MINIMUM (=R), current is MAXIMUM. Students often say both are minimum.
  • RMS ≠ Average. These are numerically different — don't confuse in calculations.

4 Qs▼
Priority
4/5 — Foundational. Most questions are direct substitution.

Magnetic Effects of Current
2–3 QsNEW▼
Priority
3/5 — Direction questions (Fleming's rule) come often.
F=qvBsinθ · r=mv/qBLorentz force on charge. Circular radius ∝ momentum.
F=BILsinθ · B_wire=μ₀I/2πr · B_solenoid=μ₀nIForce on wire. Field around wire. Field inside solenoid.
Torque τ=BINAsinθ (max at θ=90°)Moving coil galvanometer principle.
🧠 Fleming's Rules

Left hand (motor/force on conductor): Forefinger=B, Middle=I, Thumb=F. Right hand (generator/induced current): same layout but for induced EMF direction.

⚠ Common Traps
  • Left hand = force on conductor (motor). Right hand = induced current (generator). Using wrong hand = wrong direction.
  • Stationary charge in magnetic field: F=qvBsinθ. If v=0, F=0. Magnetic force needs motion.

Semiconductors, Logic Gates & Communication
2–3 QsNEW▼
Priority
4/5 — Logic gates in 90% of papers. Free marks.

N-type: pentavalent dopant (P,As) → excess electrons. P-type: trivalent (B,Al) → excess holes.

Forward bias: p→+ve terminal. Current flows. Reverse bias: p→−ve. Only leakage current.

AND:A·B · OR:A+B · NOT:ĀNAND:(A·B)̄=Ā+B̄ · NOR:(A+B)̄=Ā·B̄ · XOR:AḂ+ĀB
NAND and NOR are UNIVERSAL gatesAny other gate can be built using only NAND gates (or only NOR gates).
AM bandwidth=2fm · Range d=√(2Rh)μ=Am/Ac≤1 for no distortion. R=Earth radius, h=antenna height.
⚡ Practice MCQ
NAND gate output when both inputs are 1:
A) 1
B) 0
C) Both 0 and 1
D) Depends on voltage
NAND=NOT(AND). AND(1,1)=1. NOT(1)=0. NAND gives 0 ONLY when both inputs are 1. All other combinations → 1.
Build truth tables mentally: for 2-input gate, 4 rows (00,01,10,11). 20 seconds. Never wrong.

Electrochemistry — Cell EMF, Nernst & Faraday

Organic Chemistry — Named Reactions & Tests

Aldol: two aldehydes/ketones → β-hydroxy carbonyl. Reagent: dil. NaOH.

Cannizzaro: aldehyde with no α-H → acid + alcohol. Reagent: conc. NaOH.

Grignard (RMgX): reacts with carbonyl → adds R → alcohol after hydrolysis. Solvent: dry ether.

SN1: tertiary carbon, polar protic, racemization. SN2: primary carbon, polar aprotic, inversion.

Markovnikov: H adds to C with MORE H atoms (electrophilic addition to alkene).

Hofmann: RCONH₂+Br₂/NaOH → RNH₂. Product has ONE FEWER carbon.

Tests: Aldehyde→Tollen's(silver mirror)+Fehling's(brick red). Ketone fails both. Alcohol→Lucas test.

⚠ Common Traps
  • Aldehyde gives Tollen's. Ketone does NOT (no −H on carbonyl C). Very common conceptual trap.
  • Hofmann: product has ONE FEWER carbon than starting amide. RCONH₂→RNH₂ (not R+1 NH₂).
  • SN1=racemization, SN2=inversion. They are opposite outcomes.
